CSS Selector Rules > Naming Guide

Naming Rule

  1. Use reserved CSS selectors
  2. Use only camelCase format alphabet and numbers
  3. Don't use numbers at the first letter

Naming Rule Exception

  1. Must use PascalCase if you are javascript/AJAX developer or server-side developer
