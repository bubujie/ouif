CSS Selector Rules > Naming Guide

Naming Rule

  1. Use reserved CSS selectors
  2. Use only lowercase letters and numbers and underscore
  3. Don't use numbers or underscore at the first letter

Naming Rule Exception

  1. Must use underscore at the first letter if you are javascript/AJAX developer or server-side developer
