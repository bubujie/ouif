8. @font-face

/* FF, OP, SF, CR */
@font-face
{ font-family:ng_ttf; src:url(ng_ttf.ttf);}

/* IE */
@font-face
{ font-family:ng_eot; src:url(ng_eot.eot);}

.nanum { font-family:ng_ttf, ng_eot;} <p class="nanum"> ... </p>

ttf 파일과 eot 파일을 함께 준비한다. 비 IE 브라우저는 ttf 포멧과 otf 포멧을 지원하지만 eot 포멧을 지원하지 않는다. 반면 IE 브라우저는 eot 포멧만 지원하기 때문에 두 가지 포멧이 모두 필요하다.
